# Lume | Tela em modo claro

## Objetivo

Implementação de uma tela do projeto Lume em modo claro, aplicando os principais componentes definidos para a interface: navegação, botão, input de formulário, card, lista de tarefas e indicadores de progresso.

## Tela escolhida

Painel de estudos do Lume. A página concentra a rotina do estudante, destacando o prazo mais próximo, progresso da atividade, cadastro de nova tarefa e lista de tarefas da semana.

## Componentes implementados

- Header com navegação principal.
- Botões primário, secundário e botão neutro.
- Campos de formulário com input, select e data.
- Cards de métrica e cards de disciplina.
- Lista de tarefas com estado de prioridade.
- Indicador visual de progresso.

## Tecnologias

- HTML5.
- CSS3.
- JavaScript.

## Como visualizar

Abra o arquivo `index.html` no navegador.
